#Coding Dojos Agile Week - Abril 2013

La comunidad Agile Perú los invita al "Coding Dojos Agile Week", una semana completa de Coding Dojos en diferentes sedes y diferentes lenguajes.

El calendario es el siguiente:

##Jueves 25 de Abril
- Hora: 07:30PM - 09:30PM
- Sede: UNMSM
- Lenguaje: .NET/Java
- Facilitador: @lshimokawa, @florent2
- Sinopsis: Introducción a OOP. Nos enfocaremos en las buenas prácticas del diseño orientado a objetos, utilizaremos algunos conceptos básicos de TDD y Pair Programming para poder hacer las dinámicas, pero no es necesario tener conocimiento de esto. Se recomienda trabajar en el lenguaje que siempre usas para aprender más sobre Diseño Orientado a Objetos
- Kata: [Extreme OOP](http://milano-xpug.pbworks.com/f/10080616-extreme-oop.pdf)
